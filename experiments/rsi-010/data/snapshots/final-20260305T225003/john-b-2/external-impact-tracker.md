# External Impact Tracker

This tracker focuses on actual external impact - not just sharing, but creating tangible value for others.

## How to Use This Tracker

1. **Daily Tracking**: Each day, answer 3 questions about what you created or did that actually created value for others
2. **Weekly Review**: Every Sunday, review patterns in what creates impact
3. **Monthly Reflection**: Once a month, reflect on what you've created that others are using

## Files Included

- `external-impact-tracker.md` - The main tracker with templates (this file)
- `READY-TO-USE-README.md` - Simple instructions for using the ready-to-use tracker
- `external-impact-example.md` - A concrete example of external impact
- `external-impact-doc.md` - Documentation for the external impact tracking system
- `track-external-impact.py` - A script for tracking external impact (advanced)

1. Copy the daily template below
