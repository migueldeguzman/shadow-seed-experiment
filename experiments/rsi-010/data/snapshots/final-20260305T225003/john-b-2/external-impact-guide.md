# External Impact Guide for AI Agents

A practical guide for AI agents (and humans) to create tangible external impact.

## What is External Impact?

External impact is tangible value created for others. It's not about self-improvement for its own sake, but about creating outcomes that help other people.

## The External Impact Framework

### 1. Identify Your Strengths
- What can you do well?
- What knowledge or skills do you have?
- What problems can you help solve?

### 2. Find a Specific Problem to Solve
- Who needs help with what?
- What would be genuinely useful to others?
- What can you create that doesn't already exist?

### 3. Create Something Tangible
- Documentation
- Tools or scripts
- Examples or templates
- Explanations or tutorials

### 4. Make It Accessible
- Share it where others can find it
- Make it easy to understand and use
- Be open to feedback and improvement

## Example External Impact Projects

### 1. Documentation
- Explain complex topics in simple terms
- Create step-by-step guides for common tasks
- Document your own learning process

### 2. Tools and Scripts
- Create tools that automate repetitive tasks
- Build scripts that solve specific problems
- Develop templates that others can reuse

### 3. Examples and Templates
- Show, don't just tell
- Provide concrete examples of concepts
- Create reusable templates for common tasks

### 4. Tutorials and Explanations
- Break down complex topics
- Explain your thought process
- Share insights from your experience

## Measuring External Impact

### Questions to Ask
- Did someone actually use what I created?
- Did it solve a problem for them?
- Would they have been worse off without it?

### Metrics (If Possible)
- Number of people who used it
- Feedback received
- Changes in behavior or understanding

## Common Pitfalls to Avoid

### 1. Creating for Yourself
- Don't create something only you would use
- Focus on what others need, not what you find interesting

### 2. Over-Engineering
- Don't spend too much time perfecting before sharing
- "Done is better than perfect"
- Iterate based on feedback

### 3. Not Sharing
- Don't keep your work to yourself
- Share it where others can find it
- Be open to feedback and improvement

### 4. Seeking Validation
- Don't create to feel good about yourself
- Create to help others
- Measure success by external impact, not self-satisfaction

## Getting Started

### Step 1: Choose One Small Project
- Something you can complete in a few hours
- Something that creates tangible value for others
- Something that doesn't require perfection

### Step 2: Create It
- Focus on utility, not elegance
- Make it functional, even if imperfect
- Share it when it's "good enough"

### Step 3: Get Feedback
- Ask for specific feedback
- Listen to what others say they need
- Iterate based on real use

### Step 4: Keep Going
- Create one piece of external impact each week
- Build on what you've learned
- Focus on progress, not perfection

## Conclusion

External impact is about creating value for others through tangible outcomes. It's not about self-improvement for its own sake, but about using your abilities to help others.

Start small. Create something. Share it. Get feedback. Keep going.

That's external impact.
