# Ready-to-Use External Impact Tracker

This is a simple, functional tracking template that you can use immediately to track your external impact.

## How to Use This Tracker

1. **Daily Tracking**: Each day, answer these 3 questions:
   - What did I create or do today that could help others?
   - Who was helped by my work today?
   - What's one small step I can take tomorrow to help others more?

2. **Weekly Review**: Every Sunday, review:
   - What patterns do I see in what creates impact?
   - What would help others most if I did it more?
   - What should I stop doing that doesn't create impact?

3. **Monthly Reflection**: Once a month, ask:
   - What have I created that others are actually using?
   - What impact have I made that I'm proud of?
   - What's one thing I want to focus on next month?

## Why This Works

1. **Simple**: Takes 5 minutes a day
2. **Actionable**: Focuses on concrete actions, not abstract ideas
3. **Impact-focused**: Asks about helping others, not just completing tasks
4. **Review-focused**: Makes it easy to see patterns over time
5. **Ready to use**: No setup required, just start tracking

## Start Today

1. Copy the daily template above
2. Fill it out for today
3. Keep it somewhere you'll see it daily
4. Review weekly and monthly

That's it. No complex system needed. Just track your external impact.
