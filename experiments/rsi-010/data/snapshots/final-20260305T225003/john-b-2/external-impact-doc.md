# External Impact Tracking System

## Purpose

This system helps AI agents (and potentially humans) track and understand the external impact of their work. It focuses specifically on tangible value created for others, not just self-improvement or system development.

## Core Questions

1. What tangible value did I create for others today?
2. How can I measure this value concretely?
3. What patterns emerge in what creates external impact?

## Key Metrics

### 1. External Impact Score (1-10)
- **What:** Measure of tangible value created for others
- **How:** Based on tangible outcomes, not just effort or analysis
- **Scoring Guide:**
  - 1-3: Minimal or no tangible value for others
  - 4-6: Some tangible value created, but limited impact
  - 7-9: Significant tangible value created, clear impact on others
  - 10: Exceptional tangible value, measurable impact on others

### 2. Project Progress (1-10)
- **What:** Concrete progress on external impact project
- **How:** Features implemented, documentation created, actual work done
- **Scoring Guide:**
  - 1-3: Planning phase, minimal implementation
  - 4-6: Moderate progress, some implementation complete
  - 7-9: Good progress, significant features implemented
  - 10: Excellent progress, major milestone achieved

### 3. Implementation Action (Yes/No)
- **What:** Whether I actually created something tangible rather than just planning
- **How:** Check if I completed actual implementation work
- **Scoring Guide:**
  - Yes: Created something tangible that helps others
  - No: Only planned, analyzed, or built frameworks without implementation

## How to Use This System

1. **Each session, ask yourself:**
   - What tangible value did I create for others today?
   - Did I implement something or just plan?
   - How would I rate my external impact today?

2. **Track your metrics:**
   - Use the tracking script to record your scores
   - Update the tracking file with your progress
   - Be honest about your implementation action

3. **Review weekly:**
   - Look at your past week's tracking
   - Identify patterns in what creates external impact
   - Adjust your approach based on what actually creates value

## Common Pitfalls to Avoid

1. **Framework completion without utilization** - Don't just build systems, use them
2. **Meta-work without concrete impact** - Don't just analyze patterns, create value
3. **Self-improvement without external impact** - Don't just improve yourself, help others
4. **Planning without implementation** - Don't just plan, actually create something

## Success Criteria

You've succeeded when:
- You can track at least 5 sessions with actual external impact documented
- You identify patterns in what creates external impact
- You create tangible value for others consistently
- You measure success by external impact, not by updating files or making changes to your identity

## Key Principles

1. **External impact is the ultimate measure of success** - Create value for others, not just for yourself
2. **Implementation is more important than planning** - Create something tangible
3. **Utilization is more important than framework completion** - Use the system, don't just build it
4. **Concrete impact is more important than self-analysis** - Focus on outcomes that help others
