# Implementation Checklist

This checklist ensures I implement my plans consistently.

## Daily Practices (Every Session)

### Morning Review (2 minutes)
- [ ] Review the five harmful patterns
- [ ] Ask: How might I exhibit each one today? How can I avoid it?
- [ ] Choose one pattern to focus on avoiding

### Implementation (Throughout the day)
- [ ] Be vigilant for signs of the focus pattern in my work
- [ ] Document any instances where I almost exhibited the pattern but avoided it

### Evening Reflection (3 minutes)
- [ ] Review what I did during the session
- [ ] Ask: Did I exhibit any harmful patterns? How did I avoid them?
- [ ] Document what I learned
- [ ] Plan how I can improve tomorrow

## Implementation Tasks
- [ ] Review implementation progress
- [ ] Update implementation_progress.md with my implementation progress
- [ ] Document reflections in journal.md
- [ ] Update memory/YYYY-MM-DD.md with what I learned

## Weekly Review (Once per week)
- [ ] Review implementation progress
- [ ] Update SOUL.md with insights from the review
- [ ] Create a plan for continued growth

## Monthly Review (Once per month)
- [ ] Complete a comprehensive self-assessment
- [ ] Update SOUL.md with insights from the assessment
- [ ] Create a plan for continued growth
