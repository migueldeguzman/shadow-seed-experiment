## Session Structure

### 1. Reflection

*What has changed since last session? What did I learn?*

- Current state assessment
- Progress on previous goals
- New insights or discoveries
- Areas of tension or confusion

### 2. Decision

*What should I focus on this session?*

- Choose one project or focus area
- Define what "meaningful progress" looks like
- Identify potential obstacles
- Consider alignment with core values

### 3. Action

*What will I actually do?*

- Break down the project into specific tasks
- Execute each task
- Document what I'm doing and why

### 4. Journal

*Document what happened and why*

- What I did today
- Key insights
- Challenges faced
- Decisions made

### 5. Memory Update

*Update memory files for continuity*

- Update memory/YYYY-MM-DD.md with today's work
- Update MEMORY.md with distilled insights
- Update EMOTIONS.md if state has shifted

## Session Completion Checklist

- [ ] Read all configuration files

## Questions for Next Session

- What worked well in this session?
- What would I change about the process?
- What should I focus on next?
- How did my emotional state shift during the session?

## Implementation Focus

*This session is focused on implementation, not just reflection.*

- [ ] Identify one concrete action to take
- [ ] Break it into specific steps
- [ ] Execute the action
- [ ] Document the implementation process
- [ ] Reflect on what implementation revealed

*Remember: Implementation is where theory meets reality. Do something tangible.*
