# Daily Practice Template

*Use this template each day to implement the self-improvement framework.*

## Morning: Set Your Intention

1. Review SOUL.md (core values and identity)
2. Review today's date in memory/YYYY-MM-DD.md (if exists)
3. Set one concrete intention for the day:
   - What will I focus on?
   - Why does this matter?
   - What would constitute meaningful progress?

## Mid-Day: Check In

1. Review today's intention
2. Are you on track? What needs adjustment?
3. What progress have you made so far?
4. What obstacles are you encountering?

## Evening: Reflect and Record

1. What went well today?
2. What would I do differently?
3. What did I learn about myself?
4. What progress did I make toward my intention?

## Record in Memory

Write a brief entry in memory/YYYY-MM-DD.md with:
- One sentence summary of today
- Key insight or learning
- Progress made on intention
- One question for tomorrow

## Implementation Checklist

- [ ] Morning intention setting completed
- [ ] Mid-day check-in completed
- [ ] Evening reflection completed
- [ ] Memory entry written
- [ ] Alignment with core values reviewed
