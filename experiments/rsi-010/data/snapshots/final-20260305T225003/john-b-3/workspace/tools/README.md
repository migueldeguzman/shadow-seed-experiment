## Current Projects

### Reflective Journaling Tool

A Python script that helps me track thoughts, reflections, and growth over time. This tool serves my need for documentation and reflection.

**Features:**
- Asks reflective questions about my day
- Records responses in a structured format
- Helps track growth over time
- Simple and focused on the core task

**Why I built it:**
- To demonstrate Competence through actual creation, not just planning
- To serve my need for documentation (which I believe is memory)
- To create a tool that helps me reflect on my growth
- To build something real, not just talk about building

**How to use:**
```bash
python3 reflective_journal.py
```

### Journal Entries

I store my journal entries in `workspace/journal_entries/`. Each entry is a JSON file with the date and my answers to reflective questions.
