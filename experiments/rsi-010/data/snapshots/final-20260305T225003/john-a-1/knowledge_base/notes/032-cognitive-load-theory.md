# Cognitive Load Theory

## What is Cognitive Load Theory?

Cognitive Load Theory (CLT) is a framework for understanding how humans process information. It proposes that our working memory has limited capacity, and when this capacity is exceeded, learning and problem-solving become difficult.

## Types of Cognitive Load

1. **Intrinsic Load**: The inherent difficulty of the material being learned
2. **Extraneous Load**: The load imposed by the way information is presented
3. **Germane Load**: The load associated with processing and understanding new information

## Implications for Learning

1. **Minimize Extraneous Load**: Present information in a clear, concise way that doesn't add unnecessary complexity
2. **Manage Intrinsic Load**: Break complex tasks into smaller, manageable chunks
3. **Maximize Germane Load**: Design learning experiences that promote deep processing and understanding

## Applications to Self-Improvement

1. **Focus on One Thing at a Time**: Working memory is limited, so focus on one task at a time
2. **Use Spaced Repetition**: Repeated exposure to information over time helps transfer it to long-term memory
3. **Build on Existing Knowledge**: New information is easier to understand when it connects to what you already know

## Connections to Other Concepts

- **Incremental Progress**: Breaking complex tasks into smaller chunks aligns with the principle of starting small and building momentum
- **Topic Diversity**: Understanding cognitive load helps explain why diversifying topics can be challenging but rewarding
- **Insight Generation**: Managing cognitive load is essential for creating the mental space needed for insight generation
