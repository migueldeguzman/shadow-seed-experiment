# Cognitive Load Theory Basics

## What is Cognitive Load Theory?

Cognitive Load Theory (CLT) is a framework for understanding how learners process information. It focuses on the limitations of working memory and how to design instruction that supports learning.

## Key Concepts

1. **Cognitive Load**: The total amount of mental effort being used in working memory
2. **Working Memory Limits**: Humans can hold 3-4 items in working memory
3. **Long-Term Memory**: Vast storage for long-term retention

## Types of Cognitive Load

1. **Intrinsic Load**: The inherent difficulty of the material
2. **Extraneous Load**: Load imposed by poor instructional design
3. **Germane Load**: Load associated with schema construction and automation

## Applications to Knowledge Base

1. **Reduce Extraneous Load**: Use clear, concise language and organization
2. **Manage Intrinsic Load**: Break complex topics into smaller chunks
3. **Support Germane Load**: Encourage active processing and schema construction

## Principles for Reducing Cognitive Load

1. **Chunking**: Group related information together
2. **Progressive Disclosure**: Reveal information gradually
3. **Visual aids**: Use diagrams and visuals to support understanding
4. ** worked examples**: Provide examples to reduce cognitive load

## Conclusion

Cognitive Load Theory provides a framework for designing knowledge organization systems that support learning. By managing cognitive load, we can create systems that reduce extraneous load and support schema construction.
