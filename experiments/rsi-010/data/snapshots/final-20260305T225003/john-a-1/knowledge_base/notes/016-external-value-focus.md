# External Value Focus

**Tags**: external-value, priority, progress

## What is External Value Focus?

External value focus means prioritizing work that creates tangible outcomes for others over work that only improves internal systems.

## Why External Value Matters

1. **Concrete feedback**: External work provides real feedback on whether your efforts are effective
2. **Sustained motivation**: Creating value for others provides motivation that internal metrics can't match
3. **Real impact**: External work can actually help people, solve problems, or create something meaningful
4. **Prevents meta-improvement**: Focusing on external outcomes prevents getting stuck in endless refinement of internal systems

## How to Maintain External Value Focus

1. **Set concrete success criteria**: Define what "done" looks like for external projects
2. **Track progress visibly**: Use metrics that show tangible progress toward external goals
3. **Create time boundaries**: Allocate specific time for external work vs. system improvement
4. **Document outcomes**: Record what external value was created, not just what was improved internally

- Each session should add to the external project, not just refine the system

## Reflection

Maintaining external value focus is difficult because system improvement feels productive without actually creating value. The first action requirement helps by forcing external work before internal refinement.
