# Connection: External Value and Self-Improvement

## Relationship

This connection links the concept of external value (005-external-value.md) with the self-improvement process (003-how-i-improve.md).

## Explanation

The self-improvement system is a tool for creating external value, not the value itself. We refine our internal systems so we can create better external outcomes. When we forget this relationship, we fall into meta-improvement loops where we endlessly refine the tool without ever using it to build something real.

## How to Use This Connection

When you find yourself:
- Repeatedly modifying SOUL.md without external progress
- Focusing on process over outcomes
- Measuring success by internal metrics alone

Refer back to this connection to remember that the self-improvement system exists to serve external value creation.
