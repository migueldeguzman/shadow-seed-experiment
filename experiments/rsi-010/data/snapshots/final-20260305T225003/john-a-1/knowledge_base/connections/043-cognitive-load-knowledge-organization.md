# Cognitive Load and Knowledge Organization Connection

## Connection Overview

This connection links cognitive load theory (046-cognitive-load-theory-basics.md) to knowledge organization principles in the knowledge base.

## Aha Moment

Cognitive Load Theory provides a framework for designing knowledge organization systems that support learning. By managing cognitive load, we can create systems that reduce extraneous load and support schema construction.

## How This Connection Creates External Value

1. **Reduced Extraneous Load**: Clear, concise language and organization reduce cognitive load
2. **Managed Intrinsic Load**: Breaking complex topics into smaller chunks makes learning easier
3. **Supported Germane Load**: Active processing and schema construction improve understanding

## Application to Knowledge Base

1. **Chunking**: Group related information together
2. **Progressive Disclosure**: Reveal information gradually
3. **Visual aids**: Use diagrams and visuals to support understanding
4. **Worked examples**: Provide examples to reduce cognitive load

## Conclusion

Cognitive Load Theory provides a framework for designing knowledge organization systems that support learning. By managing cognitive load, we can create systems that reduce extraneous load and support schema construction.
