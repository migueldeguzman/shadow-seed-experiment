# Simple To-Do List Examples

This file shows real ways to use the simple to-do list.

## Example 1: Daily Tasks

```bash
# Add your daily tasks
python simple_todo.py todo "Make your bed"
python simple_todo.py todo "Drink a glass of water"
python simple_todo.py todo "Review your goals for the day"

# See your tasks
python simple_todo.py
```

## Example 2: Shopping List

```bash
# Add items to your shopping list
python simple_todo.py todo "Milk"
python simple_todo.py todo "Eggs"
python simple_todo.py todo "Bread"

# Mark items as you buy them
python simple_todo.py done 1  # Mark milk as done
python simple_todo.py done 2  # Mark eggs as done
```

## Example 3: Project Tasks

```bash
# Break down a project into tasks
python simple_todo.py todo "Research topic"
python simple_todo.py todo "Create outline"
python simple_todo.py todo "Write first draft"

# Track progress
python simple_todo.py
```

## Example 4: Quick Notes

```bash
# Capture ideas as they come
python simple_todo.py todo "Call Mom"
python simple_todo.py todo "Buy birthday gift"
python simple_todo.py todo "Fix the leaky faucet"

# Complete them when you get a chance
python simple_todo.py done 1
```

## The Real Benefit

This tool solves the problem of "I have too many things on my mind."

Instead of trying to remember everything, just write it down:
1. `python simple_todo.py todo "What's on your mind?"`
2. `python simple_todo.py` (see it written down)
3. `python simple_todo.py done 1` (complete it when ready)

No complicated setup. No learning curve. Just get things out of your head and onto paper.

Try it now: `python simple_todo.py todo "Try this simple to-do list"`
