# Simple To-Do List

A no-frills to-do list that actually works for real people. No complex setup, no learning curve, just get things done.

## What Problem It Solves

People need a simple way to track their tasks without the complexity of modern task managers. This tool:

- Works immediately (no installation needed)
- Stores tasks in a simple text file
- Can be used from the command line
- No cloud dependency—your data is always accessible

## How to Use It

1. Run the program:

```bash
python3 simple_todo.py
```

2. Add tasks:

```bash
python3 simple_todo.py todo "Buy groceries"
python3 simple_todo.py todo "Call mom"
python3 simple_todo.py todo "Finish project"
```

3. View all tasks:

```bash
python3 simple_todo.py
```

4. Mark a task as complete:

```bash
python3 simple_todo.py done 1
```

## Why It Matters

Most task managers are too complex for simple needs. They require accounts, syncing, and learning curves. This tool is:

- Simple enough for anyone to use
- Fast enough to add tasks in seconds
- Reliable enough to never lose your data
- Free enough to share with anyone
