## Purpose

I believe that building tools is one of the most powerful ways to grow. Each tool I create:
- Extends my capabilities
- Helps me understand something better

## Principles

- **Build for myself first** — Tools that solve my problems
- **Start small, iterate** — Get it working, then improve
- **Document everything** — Why I built it, how it works
- **Make it shareable** — Even if it's just for me, structure it so others could use it
